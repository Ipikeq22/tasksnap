# 🌐 Tasksnap Official Website
以 Python Streamlit 開發的官方網站，作為 Tasksnap 專題的資訊展示入口。內容包含專題介紹、功能說明、開發團隊、人才招募與贊助資訊，呈現品牌完整面貌。

---

## 🚀 功能 Features
- **專題理念展示**：介紹 Tasksnap 的核心精神與服務目標  
- **介面展示頁**：呈現 App UI Prototype 的主要截圖  
- **開發人員資訊**：團隊成員介紹、照片與作品集連結  
- **贊助方案**：簡易贊助方式與 QR Code  
- **人才招募**：以輕量互動方式呈現招募訊息  
- **回饋表單**：訪客可直接提交意見或建議

---
